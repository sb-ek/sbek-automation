SBEK Automation — Setup Guide

Step-by-step instructions to get all your API keys and credentials configured. Print this page or save as PDF (Ctrl+P).

1 WhatsApp Business API NEEDED

Time: ~20-30 minutes (one-time setup)

Phone Number ID
A numeric ID assigned by Meta to your WhatsApp Business number (NOT your actual phone number)
  1. Go to developers.facebook.com
  2. Click My Apps (top right) → Select your app (or create one: type Business)
  3. In the left sidebar, click WhatsApp → API Setup
  4. Under "From" you'll see your phone number and below it the Phone Number ID
  5. Copy the numeric ID (e.g. 123456789012345) and paste it in the dashboard
If you don't have a Meta developer app yet: Go to developers.facebook.com/apps/create → Choose Business type → Add the WhatsApp product.
Permanent Access Token
A long-lived token so the system can send WhatsApp messages on your behalf. The temporary token from API Setup expires in 24 hours — you need a permanent one.
  1. Go to business.facebook.com/settings/system-users
  2. Click Add → Name: "SBEK Bot" → Role: Admin → Create
  3. Click on the system user you just created
  4. Click Generate New Token
  5. Select your WhatsApp app from the dropdown
  6. Check these permissions:
    • whatsapp_business_messaging
    • whatsapp_business_management
  7. Click Generate Token
  8. Copy the token and paste it in the dashboard
This token does NOT expire. Keep it safe — anyone with this token can send WhatsApp messages from your number.

2 OpenRouter API Key NEEDED

Time: ~5 minutes

OpenRouter API Key
Powers all AI text generation — product descriptions, SEO meta tags, FAQs, social captions, and competitor analysis.
  1. Go to openrouter.ai
  2. Click Sign Up (or Sign In if you have an account)
  3. Go to openrouter.ai/keys
  4. Click Create Key → Name: "SBEK" → Create
  5. Copy the key (starts with sk-or-...) and paste it in the dashboard
Adding credits: Go to openrouter.ai/credits → Add funds (pay-as-you-go). Typical cost: ~$0.10-0.50 per product processed. $10 credit will last hundreds of products.

3 Postiz — Social Media Posting NEEDED

Time: ~10 minutes

Postiz API Key
For auto-posting product creatives to Instagram, Facebook, LinkedIn, Twitter, and Pinterest.
  1. Go to app.postiz.com
  2. Sign Up for an account
  3. Connect your social media accounts:
    • Instagram Business account
    • Facebook Page
    • LinkedIn (optional)
    • Twitter/X (optional)
    • Pinterest (optional)
  4. Go to Settings → API section
  5. Click Generate API Key
  6. Copy the key and paste it in the dashboard
You need to connect at least one social account in Postiz before the API key will work for posting.

4 Email (SMTP) — Gmail Setup OPTIONAL

Time: ~5 minutes | Skip this if you want emails sent from the developer's account

Gmail App Password
If you want order confirmation and shipping emails to come from YOUR Gmail address instead of the developer's, follow these steps:
  1. Go to myaccount.google.com/security
  2. Scroll down to 2-Step Verification → Turn it ON (if not already)
  3. Go to myaccount.google.com/apppasswords
  4. App: select Mail
  5. Device: select Other (Custom name) → type "SBEK"
  6. Click Generate
  7. Copy the 16-character password shown (e.g. abcd efgh ijkl mnop)

Then enter in the dashboard:

SMTP Hostsmtp.gmail.com
SMTP Port587
Email AddressYour Gmail (e.g. yourname@gmail.com)
PasswordThe 16-character App Password from step 7
From Display NameSBEK <yourname@gmail.com>
Do NOT use your regular Gmail password — it won't work. You must use the App Password generated above. If you don't see the App Passwords option, make sure 2-Step Verification is enabled first.
